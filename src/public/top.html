STAR WARS API

STAR WARSのキャラクターに関するAPIです
メソッド エンドポイント 説明
GET /api/people 全キャラクターの全情報を取得
GET /api/people/name 全キャラクターの名前を取得
POST /api/people bodyに記載されたキャラクターを登録
PATCH /api/people/update/＜name＞ ＜name＞で指定したキャラクターをbodyに記載された内容で更新
DELETE /api/people/delete/＜name＞ ＜name＞で指定したキャラクターを削除
POST /api/people/initdb DBを初期化
